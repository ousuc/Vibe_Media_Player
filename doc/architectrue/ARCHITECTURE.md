# 音频播放器 - 架构设计文档


---

## 1. 概述

### 1.1 项目简介
一个基于 Qt 的跨平台音频播放器，支持 MP3、WAV、FLAC 格式。

### 1.2 技术栈
| 层级 | 技术 | 版本 |
|------|------|------|
| 前端 | QML / Qt Quick | 5.12+ / 6.0+ |
| 后端 | C++ | C++11 |
| 构建 | CMake | 3.16+ |
| 音视频 | Qt Multimedia | 5.12+ / 6.0+ |

### 1.3 设计目标
- [ ] 跨平台（Linux/Windows/macOS）
- [ ] 支持常见音频格式
